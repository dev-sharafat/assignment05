Question-1: What are the different ways to select an element in the DOM?

DOM can be selected in various ways in JavaScript. DOM can be selected using tagName, Id, ClassName, quarySelector, quarySelectotAll, etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerhtml, innertext and textContent is: Through innerHtml we get the content inside an element with Html tag and parse the Html tag. Through innertext we get all the visible content inside an element. Using textContent we get all the visible and invisible text inside the element.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a lot like event bubbling. Instead of adding an event listener to each child separately, you can add an event listener to the parent and dynamically add or remove elements based on the event listener's command.

Question-4: What is event bubbling in the DOM?

Event bubbling routes an event to its target. This is how it works: When an element (such as a button) is clicked, an event is routed to that element. If an event handler is set for the element, the event handler is triggered. The event then "bubbles" to the parent element. If an event handler is set for the parent, this event handler is triggered. The event continues to bubble until it reaches the top (the document element). We can use stopPropaation() to stop this. This will stop the bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create, add or remove an element using JavaScript. We can add an element using appendChild(). We can create an element using createElement(). We can also remove an element using removeChild(). In this way, we can create, add or remove an element in JavaScript.
